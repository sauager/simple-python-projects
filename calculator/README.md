# Simple Python Calculator

A basic Python console calculator that performs addition, subtraction, multiplication, and division.

## How to Use
1. Run the program: `python calculator.py`
2. Choose an operation from the menu by typing the corresponding number:
   - 1 → Addition
   - 2 → Subtraction
   - 3 → Division
   - 4 → Multiplication
   - 5 → Exit
3. For operations 1–4, enter two numbers separated by space (e.g., `5 3`)
4. The program will show the result. You can perform multiple calculations until you choose to exit.

## Features
- Handles addition, subtraction, multiplication, and division
- Checks for division by zero and shows an error message
- Validates user input to prevent crashes
- Easy-to-use console menu
